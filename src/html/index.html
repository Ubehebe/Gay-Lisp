What is Lisp?

One of the foundational languages of the computer age. First described in 1960, it has influenced generations of language design down to the present, including "big" languages (Java), dynamic languages popular in web development (JavaScript, Python, Ruby), and advanced functional languages (Haskell, Scala, Clojure).

What is Scheme?

A dialect of Lisp, first described in 1975. It adds several powerful facilities to the core semantics of Lisp, and its formal but succinct specification has made it one of the most widely implemented computer languages.

Really the first?

Yes.

Software claiming to be implementations of Scheme in JavaScript is abundant. Most provide the core procedure-evaluation semantics, but fail to provide major features that Scheme requires: hygienic macros, first-class continuations, true tail recursion, and so on.

Also, no JavaScript Scheme implementation I am aware of uses a standard library (macros and procedures) written in Scheme, a fact that ought to decrease confidence in their correctness.

Why?

Because it was there.

How gay?

The penis (B==D) is Turing's normal-order Y combinator. Note that this combinator is difficult to express in most mainstream (applicative-order) languages, but is easy with Scheme macros.
